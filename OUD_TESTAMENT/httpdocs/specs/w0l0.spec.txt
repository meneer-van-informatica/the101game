# w0l0.html - TODO 0 - DESC: Minimal hello page + DOGS/CATS metronome

Constraints:
- Pure B/W wireframe (no external fonts)
- Mobile-first (max phone width), above-the-fold only
- No blocking dialogs; autoplay respects browser policies

UI:
- Center text only:
  - EN: "HELLO"
  - NL: "hallo"
- Subline: "TEAM: DOGS" or "TEAM: CATS" (DOGS must be uppercase)
- Small timer line: “back to index in Ns” (N counts down)
- Phone-width container

Audio:
- WebAudio metronome every 1s:
  - DOGS → TIK (higher square click)
  - CATS → tok (lower square click)
- Audio primes on first user gesture (pointerdown/keydown).
- If `localStorage.w_audio_intent` is fresh (<8s), start immediately.

Timer:
- Duration from `localStorage["timer:/w0l0.html"]` in seconds (default 10).
- Shows countdown label.
- On zero → navigate to `/index.html`.

Data:
- Append LMW block to local chain:
  - fields: {owner:"LMW", path:"/w0l0.html", lang, team, worth:1, ts}
  - Maintain genesis if empty.
  - Update `kukel_total` = chain.length

Routes:
- Arrive here from `/index.html`.
- On timeout → `/index.html`.

Acceptance:
- A1: Page shows HELLO/hallo according to `site_lang` (`en`/`nl`).
- A2: TEAM line shows DOGS/CATS from `team_choice` (default DOGS).
- A3: On first tap/keypress, metronome starts; DOGS plays TIK; CATS plays tok.
- A4: Countdown matches configured duration; auto-redirects to `/index.html`.
- A5: Local blockchain gains a new block with worth:1 and correct metadata.
- A6: No color other than black/white; layout fits phone width.
